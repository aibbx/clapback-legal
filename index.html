ClapBack Privacy Policy

Last updated: April 17, 2026

Overview

ClapBack ("we", "our", "the app") is an AI-powered reply keyboard. We take your privacy seriously. This policy explains what data we collect and how we use it.

Data We Collect

We do not collect, store, or share any personal data.

  • Clipboard content: When you use the keyboard, ClapBack reads your clipboard to detect comments you want to reply to. This content is sent directly to the AI API (Anthropic Claude) for reply generation and is not stored on our servers.
  • API calls: Reply generation requests are sent to Anthropic's API. These requests contain only the comment text and your selected mood/scene. No personal identifiers are included.
  • Settings: Your preferences (selected mood, API key, credit balance) are stored locally on your device using App Groups and are never transmitted.

Keyboard Full Access

ClapBack requests "Full Access" for the keyboard extension solely to make network calls to the AI API for generating replies. We do not log keystrokes, track typing patterns, or access any text fields beyond what you explicitly paste into ClapBack.

Third-Party Services

  • Anthropic API: Comment text is sent to Anthropic for AI processing. See Anthropic's Privacy Policy.
  • Apple StoreKit: Subscriptions and purchases are handled entirely by Apple. We do not process payments.

Data Retention

We retain no user data. All processing happens in real-time, and no comment text or generated replies are stored after the session ends.

Children's Privacy

ClapBack is not intended for children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this policy from time to time. Changes will be posted on this page.

Contact

If you have questions about this privacy policy, contact us at: x@hbb.ai

ClapBack 隐私政策

最后更新：2026年4月17日

概述

ClapBack（"我们"、"本应用"）是一款AI驱动的回复键盘。我们高度重视您的隐私。本政策说明我们收集哪些数据以及如何使用。

我们收集的数据

我们不收集、存储或分享任何个人数据。

  • 剪贴板内容：使用键盘时，ClapBack 会读取您的剪贴板以检测您想回复的评论。此内容直接发送至 AI API（Anthropic Claude）用于生成回复，不会存储在我们的服务器上。
  • API 调用：回复生成请求发送至 Anthropic 的 API。请求仅包含评论文本和您选择的心情/场景，不包含任何个人身份信息。
  • 设置：您的偏好设置（选择的心情、API 密钥、额度余额）使用 App Groups 存储在您的设备本地，永远不会传输到外部。

键盘完全访问权限

ClapBack 请求键盘扩展的"完全访问"权限，仅用于向 AI API 发起网络请求以生成回复。我们不会记录按键、追踪输入模式，也不会访问您明确粘贴到 ClapBack 之外的任何文本输入框。

第三方服务

  • Anthropic API：评论文本发送至 Anthropic 进行 AI 处理。请参阅 Anthropic 隐私政策。
  • Apple StoreKit：订阅和购买完全由 Apple 处理。我们不处理支付信息。

数据保留

我们不保留任何用户数据。所有处理实时进行，会话结束后不存储任何评论文本或生成的回复。

儿童隐私

ClapBack 不面向13岁以下儿童。我们不会有意收集儿童的数据。

政策变更

我们可能会不时更新此政策。变更将在此页面发布。

联系方式

如果您对本隐私政策有任何疑问，请联系：x@hbb.ai
